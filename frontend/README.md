# Frontend

Этот каталог содержит frontend CRM Kit на Next.js.

## Что уже есть

- главная страница
- экран входа
- dashboard-шаблон
- небольшой API-клиент для проверки backend
- общие UI-компоненты
- пример `.env.example` для адреса backend

## На что смотреть новичку

- `app/page.tsx` - стартовая страница
- `app/login/page.tsx` - экран входа
- `app/dashboard/page.tsx` - dashboard
- `lib/api.ts` - пример подключения к backend
- `components/` - переиспользуемые блоки интерфейса
- `frontend/.env.example` - пример переменных окружения для API
