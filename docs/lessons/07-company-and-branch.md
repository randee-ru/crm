# Урок 07 - Компания и филиал

## Простое объяснение

В SaaS-платформе компания - это основной клиент системы.
Филиал - это отдельная точка работы внутри этой компании.

## Зачем это нужно

Без такого разделения данные разных клиентов могут случайно оказаться в одной куче.
Это опасно и для бизнеса, и для безопасности.

## Как это устроено в CRM Kit

- `Company` - tenant
- `Branch` - дочерний объект компании
- у филиала всегда есть ссылка на компанию
- slug создаётся автоматически, чтобы у записи был удобный идентификатор

## Что проверяют тесты

- компания получает slug автоматически
- филиал получает slug автоматически
- филиал связан с правильной компанией
