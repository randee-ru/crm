# Урок 11 - Что такое основа frontend

## Простое объяснение

Frontend bootstrap - это стартовая структура frontend-приложения.
Она задаёт основной стиль, навигацию, layout и первые экраны.

## Зачем это нужно

Если начать frontend без основы, то каждая новая страница будет выглядеть и работать по-своему.
Это быстро превращается в хаос.

## Что мы собрали в CRM Kit

- главную страницу
- экран входа
- dashboard-шаблон
- API-клиент
- общие UI-компоненты

## Какой стиль выбран

Мы уходим от декоративного лендинга в сторону плотного корпоративного интерфейса:

- светлая панель управления
- постоянная боковая навигация
- верхний поиск и быстрые действия
- компактные карточки, таблицы и лента событий
