# Админка CRM Kit

Этот раздел описывает внутреннюю админку проекта.
Здесь мы показываем не HTTP API, а конфигурацию Django admin и Unfold.

## Что уже есть сейчас

## Принцип документации

Каждая админ-страница должна отвечать на четыре вопроса:

1. что делает эта сущность
2. как она выглядит в админке
3. каким тестом подтверждается регистрация
4. что можно взять как шаблон для новых моделей
