# Админка: основа Unfold

## Назначение

Этот раздел показывает, как мы подтверждаем работу админки на уровне кода.
В отличие от обычного endpoint'а, здесь мы проверяем не HTTP-ответ, а саму конфигурацию admin.

## Что именно проверяем

- админ-классы наследуются от `unfold.admin.ModelAdmin`
- модели зарегистрированы в `admin.site`
- внутренний интерфейс можно использовать как рабочий инструмент для сотрудников
- у админки есть понятный заголовок CRM Kit

## Связанный тест

`backend/core/tests/test_unfold_admin.py`
