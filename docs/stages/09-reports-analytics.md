# Этап 9 - Отчёты и аналитика

Цель:
- превратить дневной отчет в отдельный модуль отчётов
- добавить сводный analytics dashboard
- собрать ключевые метрики бизнеса в одном месте

Результаты:
- модуль `reports`
- модуль `analytics` на уровне backend API
- дневной отчет `/api/v1/reports/daily/`
- сводка `/api/v1/reports/overview/`
- frontend-экран `/dashboard/reports`

Критерии готовности:
- у владельца бизнеса есть сводка по клиентам, бронированиям, посещениям, продажам и платежам
- есть отдельная страница отчётов
- отчёты можно расширять без переписывания существующих экранов
