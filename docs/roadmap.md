# CRM Kit Roadmap

## Stage 1 - Foundation

- create repository scaffold
- document project purpose
- define folder structure
- prepare handoff context

## Stage 2 - Backend bootstrap

- create Django project skeleton
- configure settings by environment
- add Docker and Compose
- prepare basic health checks
- document the bootstrap steps for a beginner

## Stage 3 - Core domain setup

- create shared core utilities
- add first business modules
- define base entities and conventions

## Stage 4 - Frontend bootstrap

- create Next.js app shell
- connect frontend to backend
- establish UI conventions

## Stage 5 - SaaS foundation

- tenants/companies
- roles and permissions
- audit logging
- notifications
